# AppWindows

Лёгкое десктопное приложение WinForms (C#, .NET Framework 4.8) для просмотра и выполнения SQL-отчётов + CRUD-операций.

## Основные возможности
- Главное навигационное окно с кнопками для открытия форм:
  - Управление сотрудниками
  - Информация о руководителях
  - Образовательные организации
  - Номинации
  - Комбинированный просмотр данных
  - Сортировки и отчёты
  - Агрегационные/отчётные запросы
- Каждая форма содержит интерфейс и логику выполнения SQL-запросов (см. папку `Sql-queryForms`).
- Цель сборки: .NET Framework 4.8 (под Visual Studio 2022).

## Быстрый старт (для разработчика)
1. Клонировать репозиторий:
   - `git clone git@github.com:Ximelay/AppWindows.git`
2. Открыть решение в Visual Studio:
   - через __Open Solution__ (двойной клик по `.sln` или через меню).
3. Настроить строку подключения к базе данных:
   - Откройте `App.config` (или свойства проекта) и обновите секцию `connectionStrings`.
   - Пример строки подключения к SQL Server:
     - `Server=SERVER_NAME;Database=DB_NAME;User Id=USERNAME;Password=PASSWORD;`
4. Собрать и запустить:
   - __Build Solution__ (или Ctrl+Shift+B).
   - Запустить приложение: __Start Debugging__ (F5) или __Start Without Debugging__ (Ctrl+F5).

## Структура проекта (вкратце)
- `MainForms/` — главные окна навигации и точки входа UI (`MainForm.cs`).
- `AllGridData` — формы вывода каждой сущности из БД.
- `Sql-queryForms/` — формы, выполняющие SQL-запросы и отображающие результаты.
- `Properties/`, `App.config` — настройки проекта и строки подключения.
- Остальные стандартные WinForms файлы (дизайнеры, ресурсы).

## Заметки
- Проект рассчитан на Windows (WinForms).
- В коде используются возможности C# 7.3 и таргет .NET Framework 4.8.
